# Retrieve supported Chains

This endpoint will return of list of all the blockchains that are supported by the NFT API.

## Endpoint: GET

```javascript
https://api-business.arkane.network/api/env
```

## Returns

```javascript
{
    "supportedChainsForItemCreation": [
        "AVAC",
        "BSC",
        "ETHEREUM",
        "MATIC"
    ]
}
```
